# VidGo Ultra Pro

Production-ready Flutter 3+ Android project for a browser-based multi-platform video downloader.

## Package / Platform
- App name: **VidGo Ultra Pro**
- Package name: **com.mcr.vidgopro**
- Android target SDK: **34**
- Null Safety enabled

## Implemented Modules
- First launch flow with permissions + terms modal + onboarding tutorial
- Built-in browser with smart URL-based platform/video detection
- Download manager with queue, progress, speed, ETA, pause/resume/cancel
- Saved library with filtering, share, delete
- Full-screen video playback with gestures
- Clipboard URL monitoring with smart open suggestion
- Dark/light mode toggle and premium-style UI components
- Ads unit ID centralization and test mode defaults
- API contract placeholders for backend integration

## Build
1. Install Flutter 3.x
2. `flutter pub get`
3. `flutter run`

## Note
Backend endpoints are intentionally defined and left empty for later integration.
