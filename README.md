# VidGo Ultra Pro

Flutter 3+ Android-first application for browser-based multi-platform video detection, download management, offline playback, clipboard URL monitoring, theme control, and ad integration scaffolding.

## Package
- Android application ID: `com.mcr.vidgopro`
- App Name: **VidGo Ultra Pro**

## Notes
- API endpoints are defined in `lib/services/api_service.dart` and intentionally left empty for integration.
- Google Mobile Ads uses test IDs by default in `lib/ad_service.dart`.
